# Control structures

-----------------------------------------------------------------------------

# `if`

```java
if (conditional) {
  // body
}
```

-----------------------------------------------------------------------------

# `if/else`

```java
if (conditional) {
  // body
} else {
  // body
}
```

-----------------------------------------------------------------------------

# `if/else if/else`

```java
if (conditional) {
  // body
} else if (conditional) {
  // body
} else if (conditional) {
  // body
} else if (conditional) {
  // body
} else {
  // body
}
```
